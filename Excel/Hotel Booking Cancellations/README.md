# 🏨 Hotel Booking Cancellation Analysis Dashboard

## 📌 Project Overview
This project analyzes hotel booking data to understand booking patterns, cancellations, guest types, and seasonal trends.  
The dashboard was created using **Microsoft Excel, Power Query, Pivot Tables, and Charts** to transform raw data into meaningful insights.

This project demonstrates skills in **data cleaning, data analysis, and dashboard design using Excel**.

---

## 📊 Dashboard Features

- 📅 Year-wise filtering using slicers
- 👥 Guest type analysis (Couple, Family, Single)
- 📈 Monthly booking vs cancellation trend
- 🏨 City Hotel vs Resort Hotel comparison
- 🛏 Room type booking vs cancellation analysis
- 📊 Interactive dashboard using Pivot Charts & Slicers

---

## 🛠 Tools & Technologies Used

- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Visualization

---

## 📂 Dataset Information

The dataset contains hotel booking records including:

- Arrival year
- Guest type
- Hotel type (City Hotel / Resort Hotel)
- Room status
- Booking status
- Cancellation status
- Monthly booking data

Dataset used for learning and analysis purposes.

---

## 📊 Key Insights

- Couples made the highest number of bookings
- City hotels received more bookings than resort hotels
- Highest bookings occur between May – August
- Cancellation count is significant for desired rooms
- Booking trends change based on season
