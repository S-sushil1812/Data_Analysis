# 📊 Adventure Works Sales Analysis Dashboard

## 📌 Project Overview

This project is an interactive Sales Analysis Dashboard built using Microsoft Excel, Power Query, Pivot Tables, Charts, and VBA Macros.
The dashboard provides insights into sales performance, profit distribution, customer contribution, and product analysis using the Adventure Works dataset.

The goal of this project is to transform raw sales data into meaningful insights through data cleaning, data modeling, and visualization.

## 🚀 Features

📈 Time Analysis Dashboard

📊 Detailed Sales Analysis Dashboard

🔍 Year, Month, Country Filters

🧮 Profit, Revenue, COGS, Transactions KPIs

👥 Customer & Product Profit Analysis

🌍 Country-wise Profit Contribution Map

🎨 Product Color & Pricing Analysis

⚡ Macro Button for Clear Filter

🔄 Macro for Dashboard Navigation

🧹 Data Cleaning using Power Query


## 🛠 Tools & Technologies Used

- Microsoft Excel

- Power Query

- Pivot Tables

- Pivot Charts

- VBA Macros

- Data Visualization

- Dashboard Design

## 📊 Key Insights from Dashboard

- Top 5 products contribute significant profit share

- United States & Australia generate highest profit

- Expensive products contribute majority of profit

- Weekdays generate more profit than weekends

- Age group 60+ contributes notable profit share


## ⚙️ How to Use

- Download the Excel file

- Enable Macros

- Use filters (Year / Month / Country)

- Use navigation buttons to switch dashboards

- Use Clear Filter button to reset
